# SimArena_Client
Dieses Repository stellt euch einen Custom C#-Client zur Verfügung, mit dem ihr ganz einfach Charakter-Klassen an eine API übermitteln könnt. Dort können eure Charaktere gegen zufällige oder selbst erstellte Gegner antreten – und das ganz ohne, dass ihr selbst HTTP-Requests schreiben müsst.
